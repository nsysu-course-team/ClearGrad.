📖 專案簡介 (Overview)
ClearGrad 是一個解決大學生「畢業學分計算」痛點的 Web 應用程式，透過直覺的視覺化介面以及自動化計算，協助學生追蹤必修、選修、通識（包含博雅向度）以及跨域學習 （各類學程）／國際學習的修課進度。
本專案採用 GPL-3.0 開源授權，並結合人機協作（AI-Augmented Development）模式開發。

✨ 核心功能 (Key Features)
* 🎯 完整學分追蹤：支援系必修、選修、通識（D1-D6 向度）及體育學分自動統計
* 🔗 跨域與微學程：整合複雜的跨領域學程規則，一鍵檢核畢業門檻
* 📊 GPA 計算機：支援 4.30 制 GPA 即時試算
* 💾 資料安全：採用 LocalStorage 技術，支援資料即時保存、備份與還原
* 📱 響應式設計：完美支援手機與電腦版瀏覽

🛠️ 技術架構 (Tech Stack)
* Frontend: HTML5, CSS3, JavaScript (Vanilla)
* Data Storage: LocalStorage API
* Development: AI-Assisted (Google Antigravity), GitHub Pages
* Design: Custom UI/UX
